CornerStone蛋疼问题

小坑环境

OS X: 10.11.4
CornerStone: v2.7.18
SVN 版本：v1.6

小坑描述

最近使用CornerStone提交项目出现小坑，添加文件夹时出现错误提示：

Description: ... containing working copy admin area is missing
Status:155005
File:subversion/libsvn_wc/lock.c,947

一种填坑方法

第一次遇到这个问题时，是提交整个下午的代码，晚上8点提交报错，当时就懵得晕乎，几经折腾，终于找到一种结局方法：

进入Finder复制出错的文件夹到别处，CornerStone中出现如下提示：

然后点击右键，选择 delete :

再将文件夹复制到原来的地方，可直接在Finder使用 command + Z，随后CornerStone出现如下提示：

接着正常Add并提交项目即可。
