TableViewCell的自定义按钮定位

LongerONE 13 Jul 2016

在iOS开发中，TableView中的点击事件通常是以indexPath.row来处理，但是，如果TableViewCell中有多个自定义操作，如何确定点击了哪一行的哪一个按钮。

一种方案：

  • 先为按钮添加点击方法： 
     [cellC.btn addTarget:self action:@selector(clickCellButton:withEvent:) forControlEvents:UIControlEventTouchUpInside];
  • 再通过touch事件定位到具体的位置：
  - (void)clickCellButton:(UIButton *)sender withEvent:(id)event {
    //获取点击所在的 IndexPath
    NSSet *touches = [event allTouches];
    UITouch *touch = [touches anyObject];
    //获取在列表的点击坐标
    CGPoint touchPoint = [touch locationInView:self.myTableView];
    NSIndexPath *indexPath = [self.myTableView indexPathForRowAtPoint:touchPoint];
    
    if (indexPath != nil) {
        //可通过tag再进行具体判断
        
  	 }
  }
